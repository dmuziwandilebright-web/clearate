Financial Truth, Simplified.

Clearate is designed to give you clarity on Zimbabwe's exchange rates. Use this guide to master the tools available.

Price Check helps you verify if a retailer's price matches official rates:

  1. Enter the USD Price of the item.
  2. Enter the ZAR or ZiG Price being charged by the shop.
  3. Clearate will automatically compare the two using current RBZ data.
check_circle Fair — The price is within the legal limit.
warning Overcharged — The rate exceeds official guidelines.
info Undervalued — The price is lower than expected.

Clearate uses QR Sync for secure, offline sharing between devices:

  • qr_code_2 Show QR: Generate a code on a phone with updated rates.
  • qr_code_scanner Scan: Use another phone's camera to read the code and sync data.

This method is safe, instant, and requires zero data usage or cellular network.

Clearate values transparency. All exchange rate data is sourced directly from the Reserve Bank of Zimbabwe (RBZ) via the authorized ZimRate API.
